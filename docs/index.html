Custom Product Format Picker

Create your Custom Product Metafield Meta

  1. Logon to your Shopify Admin page
  2. Go to Settings -> Custom Data -> Products
  3. click "Add Definition"
  4. Name = linkedhandles
  5. Click "Select Type" -> Single Line Text
  6. Select "List of Values
  7. Click "Save"

Set the Product Type

  1. Logon to your Shopify Admin page
  2. Go to Products -> The Product you want to modify
  3. Under "Product Organization" (The left column) Enter the "Product Type" E-book for example. This is what will appear in the format button text
  4. Hit "Save"

What is the Product Handle?

The Product Handle is simply the friendly part of the URL that descirbes your product. It can be found in two places.
  1. In the url for the product on your Shopify Site. For example: https://yoursite.com/products/this-part-is-the-handle
  2. OR..The URL shown in the Search Engine Listing in your product page on the Admin console.

Set the linkedhandles Metafield Values

  1. Logon to your Shopify Admin page
  2. Go to Products -> The Product you want to modify
  3. Scroll down to the linkedhandles metafield
  4. Enter the handle(s) of the product formats that you like to link to the current product
