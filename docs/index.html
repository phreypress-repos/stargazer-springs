Create your Custom Product Metafield Meta

  1. Logon to your Shopify admin page
  2. Go to Settings -> Custom Data -> Products
  3. click "Add Definition"
  4. Name = linkedhandles
  5. Click "Select Type" -> Single Line Text
  6. Select "List of Values
  7. Click "Save"
